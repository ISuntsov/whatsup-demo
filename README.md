## Prerequisites

- Node.js
- npm (comes bundled with Node.js)
- Git

## Installation

1. Clone the repository:
git clone https://github.com/ISuntsov/whatsup-demo.git

2. Install dependencies:
npm install

## Running the Project

1. Start the development server:
npm run dev

2. Open the browser on http://localhost:5173/
